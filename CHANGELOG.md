# Changelog

All notable changes to `blogr-artist` will be documented in this file.

## v1.0.0 - 2026-06-29

### ✨ Features

- **Portfolio**: full portfolio system with artworks, categories, and tags (index + detail pages)
- **Commissions**: commission management with responsive carousel, pricing plans, and status tracking
- **Status field**: replace `is_featured` with flexible `status` field (draft/published/featured)
- **LinkTypeRegistry**: implement `RegistersLinkTypes` trait for Blogr's link type system
- **Per-page URL routes**: dedicated routes for portfolio and commission pages
- **Artist settings**: Filament settings page for portfolio configuration
- **Admin UI**: Filament resource for artwork management with translations
